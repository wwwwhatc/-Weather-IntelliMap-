
使用 requests 从 OpenWeatherMap API 获取当前天气数据和天气预报数据，支持多个城市的查询。
数据包括温度、湿度、风速、气压等。

可视化：使用 Matplotlib 和 Cartopy 绘制全球天气地图，标记多个城市的位置，显示温度。
为不同城市生成数据图表（如温度、风速等）。使用 Tkinter 创建 GUI 界面，允许用户输入城市、选择天气类型并查询。

天气描述：提供了天气描述的中文翻译（如晴天、阵雨等）。
